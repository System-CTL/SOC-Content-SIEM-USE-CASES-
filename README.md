# SOC-Content-SIEM-USE-CASES-
SOC-Content (SIEM USE-CASES) is a curated repository of ready-to-deploy SIEM detection use-cases, designed for SOC analysts, detection engineers, and DFIR teams. The project includes:  Correlation Rules (QRadar CRE rules, Splunk SPL, Elastic/KQL, Sigma-based logic)  Parser/Field Normalization Examples  Threat Detection Logic mapped to MITRE ATT&amp;CK.
